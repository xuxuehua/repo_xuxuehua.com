# svg

Scalable Vector Graphic 



# Tags



## g 

group elements



### transform

Control group elements for moving to different location

```
<g transform="translate(0, 200)">
</g>
```



#### translate

inside function for moving

translate(x, y)

```
translate(0, 200)
```





#### scale

scale the group elements

```
<g transform="scale(1.5)">
</g>
```
