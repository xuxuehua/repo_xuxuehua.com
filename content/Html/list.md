# list 列表



## `<ul>`无序



### disc

实心圆

```
<ul type="">
```



### circle

空心园



### square

实心方块





## `<ol>`有序列表

默认会生成序号



### A

以大写字母显示排序



### a

以小写字母显示排序



### I

以大写I显示排序

### i

以小写i显示排序



### start

从指定数字开始

```
<ol start="10">
	<li>IOS</li>
	<li>Android</li>
	<li>Linux</li>
```

> 序号从10 开始



## `<li>`列表内的项

li -> List Item

li 中可以嵌套其他的子ul ol dl等





## `<dl>` 自定义列表
