# 进制

## 二进制

前缀加0b

```
In [5]: 0b10                                                                                          
Out[5]: 2

In [6]: 0b11                                                                                          
Out[6]: 3
```



## 八进制

前缀加0o

```
In [7]: 0o10                                                                                          
Out[7]: 8

In [8]: 0o11                                                                                          
Out[8]: 9
```



十六进制

前缀加0x

```
In [14]: 0x10                                                                                         
Out[14]: 16

In [15]: 0x1F                                                                                         
Out[15]: 31
```
