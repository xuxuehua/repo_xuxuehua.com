# Google 



## allintitle

返回页面标题中包含多组关键词的文件

```
allintitle:zabbix hadoop
```

> 返回既包含zabbix也包含hadoop的页面





## intitle

返回页面title中包含关键词的页面

```
intitle:zabbix
```

> 只对后面的第一个关键词生效





## inurl

搜索查询词出现在url中的页面

```
inurl:hadoop
```

```
inurl:gov 房价
```



## filetype

指定文件格式

```
filetype:pdf hadoop
```





## site

搜索域名下面的所有文件

```
linux site:www.xuxuehua.com
```
