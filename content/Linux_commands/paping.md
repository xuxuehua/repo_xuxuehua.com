# paping



## Installation



```
wget https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/paping/paping_1.5.5_x86-64_linux.tar.gz

tar zxvf paping_1.5.5_x86-64_linux.tar.gz
```



## -p, --port N 

指定被测试服务的 TCP 端口（必须）



## --nocolor 

屏蔽彩色输出



## -t, --timeout	

指定超时时长，单位为毫秒，默认值为 1000



## -c, --count N	

指定测试次数
