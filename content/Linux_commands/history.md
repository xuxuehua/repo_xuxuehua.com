# history





## -c clean

Run the following linux command to clean both history file and all currently history file unsaved commands

```
$ history -c
```



## -d   

指定具体数目的历史并删除，还可以指定数目删除多少个



## -w

将缓冲区的文件写到文件中去



## HISTSIZE   

命令历史大小      echo $ HISTSIZE
